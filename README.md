# CryptoBot - Streamlit Algo Trader

This is a lightweight algorithmic trading dashboard using Streamlit.

### Features
- Live price polling (CoinGecko → fallback CoinPaprika)
- SMA strategy
- Buy/Sell logic
- Balance persistence
- Connection status indicator
- Multi-asset support
- Auto-refresh

### Deploy on Streamlit Cloud
1. Push this repo to GitHub.
2. Add the app on streamlit.io.
3. Ensure these two files exist:

- requirements.txt
- runtime.txt  → python-3.11

The app will then deploy without errors.
